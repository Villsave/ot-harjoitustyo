# Käyttöohje
## Konfigurointi
Ohjelma olettaa, että juurikansiosta löytyvät sen tarvitsemat kuvatiedostot.

```
food.png
pacman_1.png
pacman_2.png
pacman_3.png
pacman_4.png
ghost_0.png
ghost_1.png
ghost_2.png
ghost_3.png
brick.png
```

## Käynnistys
Ohjelman voi käynnistää komennolla
```
java -jar Pacman.jar
```
